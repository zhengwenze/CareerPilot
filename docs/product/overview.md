# Product Overview

Career Pilot 是一个连续工作流型求职工作台，不是信息展示网站。

## Core Workflow

1. 上传主简历文件
2. 提取并整理为可编辑 Markdown
3. 保存目标岗位 JD
4. 生成岗位定制版简历
5. 继续进入模拟面试
6. 在后续流程中复用已沉淀的简历、岗位和用户上下文

## Delivered Product Direction

当前产品重点在以下链路：

- 简历上传与解析
- Markdown 简历编辑与保存
- 目标岗位保存与解析
- 专属简历生成、重试与下载
- 模拟面试会话创建、继续、结束、删除、重试

## Reserved Or Incomplete Areas

这些内容不能被描述为已完成能力：

- 设置页
- 投递追踪页
- 语音相关能力
- 广义多 agent 平台化扩展

## Product Principle

所有结构设计和工程拆分都要服务于一个目标：

让用户在“简历 -> 岗位 -> 定制版简历 -> 面试训练”之间自然连续，而不是在多个彼此独立的页面和数据源之间来回切换。
