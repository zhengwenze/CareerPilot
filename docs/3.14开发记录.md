## 🔑 核心技术栈

### 后端
- **框架**：FastAPI 0.115.0
- **数据库**：PostgreSQL + Alembic
- **认证**：JWT (PyJWT)
- **验证**：Pydantic 2.10.0
- **配置**：pydantic-settings 2.6.1
- **对象存储**：MinIO
- **测试**：pytest 8.3.3

- **框架**：Next.js 15.1.0
- **UI 库**：shadcn/ui + Tailwind CSS
- **状态管理**：React Context
- **路由守卫**：自定义高阶组件
- **API 客户端**：自定义 fetch 封装
