# Docs

这一级目录只放长期有效、能被人和 Codex 反复复用的项目文档。

当前文档入口：

- `index.md`: 文档总导航，先读这里
- `codex-workspace.md`: 仓库地图，说明哪些目录是业务代码、哪些是 agent 资产、哪些只是参考素材
- `git-sync-workflow.md`: GitHub + Gitee 双仓库同步发布约定
- `mcp-setup.md`: 外部工具 / MCP 接入约定

业务知识树：

- `product/`: 产品目标与边界
- `architecture/`: 系统结构与运行流
- `domain/`: 核心业务链路与实体

新增文档时，优先放这里，而不是把结构说明堆进零散注释或临时对话里。
